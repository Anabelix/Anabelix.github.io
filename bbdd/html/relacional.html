MODELO
RELACIONAL

A partir del modelo E/R se ha diseñado el modelo relacional de tal forma que se ha conseguido un diseño más limpio de cómo se estructura el sistema, las entidades que tienen más peso y son las que poseen atributos que se van propagando al resto de entidades. La disposición final de este modelo servirá como guía para la creación de las tablas y la inserción del contenido.
